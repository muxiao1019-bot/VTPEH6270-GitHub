# VTPEH6270-GitHub
# Smoking Prevalence and Education Level in U.S. Adults

## Project Description
This project examines the association between smoking prevalence and 
education level among U.S. adults using the 2024 Behavioral Risk Factor 
Surveillance System (BRFSS) dataset.

## Author
- **Yihan Zhong**
- Program: MPH, Spring 2026
- Course: VTPEH 6270

## Research Question
Is smoking prevalence associated with education level among adults in the 
United States? Specifically, does the probability of being a current smoker 
decrease as educational attainment increases?

## Data Source
- **Dataset:** Behavioral Risk Factor Surveillance System (BRFSS) 2024
- **File:** LLCP2024.XPT (not included in repository due to file size)
- **Access:** https://www.cdc.gov/brfss/annual_data/annual_2024.html
- **Key Variables:** `_RFSMOK3` (smoking status), `EDUCA` (education level)
